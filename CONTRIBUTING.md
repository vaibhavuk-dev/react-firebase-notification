# Contributing

We welcome contributions!

## Development Setup

1. **Clone the repository:**
   ```bash
   git clone https://github.com/VAIBHAVUK-DEV/react-firebase-notification.git
   cd react-firebase-notification
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Build the package:**
   ```bash
   npm run build
   # or watch mode
   npm run build -- --watch
   ```

4. **Run the example app:**
   ```bash
   cd examples/demo-vite
   npm install
   npm run dev
   ```

## Testing

Run unit tests with Vitest:

```bash
npm test
```
